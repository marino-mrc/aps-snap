# aps-snap
The repo contains the snapcraft.yaml file used for building the snap of the [aps CLI](https://github.com/marino-mrc/aps-cli])<br />
More info on the APS project can be found [here](https://gitlab.marino-mrc.com/aps/aps-server)

## Build instructions
snapcraft --use-lxd  

### Reset the env with
snapcraft clean
